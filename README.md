# 📌 Responsive Front Page with Sidebar Toggle

A modern, fully responsive front page layout featuring a smooth and interactive sidebar toggle navigation. This project demonstrates how to build a clean UI with a collapsible sidebar that works seamlessly across desktop, tablet, and mobile devices.

# 🚀 Features

📱 Fully responsive design

🎯 Smooth sidebar open/close animation

🧭 Collapsible navigation menu

💡 Clean and minimal user interface

⚡ Lightweight and fast performance

🛠️ Built with HTML, CSS, and JavaScript

# 🛠️ Tech Stack

HTML5

CSS3 (Flexbox / Grid)

JavaScript (Vanilla JS)

# ⚙️ How It Works

Click the toggle button to open or close the sidebar.

The layout automatically adjusts for smaller screens.

Smooth transitions enhance the overall user experience.

Admin dashboards
